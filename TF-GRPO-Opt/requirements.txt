# 1. torch 与 torch_npu 的版本必须严格一致，否则 NPU 算子注册会失败。
#    常见匹配关系（以实际 CANN 版本为准）：
#      CANN 8.0.RC3  →  torch==2.1.0   + torch_npu==2.1.0.postX
#      CANN 8.0.0    →  torch==2.1.0   + torch_npu==2.1.0
#      CANN 8.1.0    →  torch==2.3.1   + torch_npu==2.3.1
#
# 2. sentence-transformers 的 Encoder 在 NPU 上运行时，
#    SentenceTransformer(..., device="npu:0") 需要 torch_npu 已正确加载。
#
# 3. 若 Qwen2.5-Math-7B-Instruct 权重未下载，可提前执行：

#    HF_ENDPOINT=https://hf-mirror.com huggingface-cli download Qwen/Qwen2.5-Math-7B-Instruct
