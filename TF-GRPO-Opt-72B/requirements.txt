#   # 4. 安装 torch_npu 2.4.0（从昇腾社区下载 aarch64 whl）

# 1. CANN / torch / torch_npu 版本对应关系（必须严格一致）：

#      CANN 8.1.x  →  torch==2.3.1  + torch_npu==2.3.1
#      CANN 8.2.RC1→  torch==2.4.0  + torch_npu==2.4.0      ← 本机适用
#      CANN 8.3.x  →  torch==2.5.1  + torch_npu==2.5.1

#    服务器 base 环境为 3.13，务必在新建的 tfgrpo (python=3.10) 中运行。
#
# 3. numpy 版本：保持 <2.0.0，torch_npu 2.4.0 与 numpy 2.x 存在兼容风险。
#
# 4. sentence-transformers 的 Encoder 在 NPU 上运行时，
#    SentenceTransformer(..., device="npu:0") 需要 torch_npu 已正确加载。
